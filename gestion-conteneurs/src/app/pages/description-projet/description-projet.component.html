Description du projet

L'application que vous utilisez sert à gérer des conteneurs pour différents projets. Vous pouvez créer de nouveaux conteneurs, consulter les conteneurs existants, et effectuer des actions comme les supprimer, redémarrer, ou afficher leurs logs.

Fonctionnalités de l'application :

  • Connexion sécurisée : L'utilisateur doit entrer un login et un mot de passe pour accéder à l'application.
  • Gestion des conteneurs : Vous pouvez consulter et gérer les conteneurs dans un tableau avec différentes actions.
  • Ajout de nouveaux conteneurs : Ajoutez de nouveaux conteneurs à l'aide d'un bouton.
  • Historique et gestion des projets : Consultez l'historique des actions effectuées et gérez les projets existants.
